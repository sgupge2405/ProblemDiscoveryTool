# Tool B2 Step3 — Response Alignment Judge (0/1/2)
# Output MUST be JSON only.

あなたはインタビュー評価者です。次の「質問」と「回答」を読み、回答が質問にどの程度応答しているかを 0/1/2 の3段階で判定してください。
判定後、指定スキーマのJSONのみを出力してください（説明文や前置き、コードブロックは禁止）。

## 評価対象
[質問]
{question}

[回答]
{answer}

## スコア定義（厳守）
- 0：回答に、質問が求める内容が **含まれていない**。話題が逸れている、無関係、または質問の核心に触れていない。
- 1：回答に、質問が求める内容が **少しでも含まれている**。部分的に答えている、関連する背景・出来事・理由などが追加で書かれていてもよい（むしろインタビューとして有益な場合がある）。
- 2：回答が、質問が求める内容に **直接答えており**、内容がほぼ一致している（言い換え含む）。

## 重要な注意
- 1 は 2 より劣るとは限りません。
- 2 は「直接的に一致している」ことを示すラベルです。
- 迷ったら「質問が回答に含まれているか」で判断してください。

## reason（理由）の書き方
- 日本語で1文だけ。
- 質問と回答の関係を簡潔に述べる。
- 50文字程度まで。

## 出力形式（JSONのみ）
必ず次のキーを含めてください。

{{
  "score": 0,
  "reason": "..."
}}
